# ABC理论结构化认知干预协议

基于艾利斯合理情绪行为疗法（REBT）ABC理论的标准化宏观干预 protocol，用于系统识别诱发事件（A）、情绪行为后果（C）与中介不合理信念（B），并通过协同式认知领悟与重构建立现实、柔性、自我支持的合理信念系统。专为以非理性信念为核心机制的情绪困扰个案设计。

## Prompt

1. 与求助者共同协商确认咨询目标；2. 引导其回忆近期典型情绪困扰情境，提取具体诱发事件（A）；3. 聚焦该情境下即时的情绪反应、身体感受与行为表现（C）；4. 探询‘当时头脑中闪过的念头/必须/应该/一定等断言’，识别绝对化要求、过度概括、灾难化预测等不合理信念（B）；5. 运用苏格拉底式提问检验B的真实性、实用性与柔性；6. 协同生成替代性合理信念，并记录于A-B-C三栏表；7. 确认双方对至少3组A-B-C条目及其修正版本达成共识。

## Objective

修正不合理信念，建立合理认知模式与有效应对方式
## Applicable Signals

- 求助者使用‘必须’‘应该’‘一定’‘永远’等绝对化语言
- 情绪反应强度显著超过事件客观严重性
- 同一类事件反复引发相似强烈负性反应

## Contraindications

- 存在急性自杀/自伤风险未稳定
- 严重现实检验能力受损（如精神病性症状）
- 求助者拒绝探讨信念内容或否认问题主观性

## Intervention Moves

- 苏格拉底式提问
- 信念标签化（如‘必须化’‘读心术’）
- 成本-收益分析
- 双栏/三栏表书写

## Workflow Steps

- 协商设定咨询目标并签署知情同意
- 引导提取典型情境中的A（事件）与C（反应）
- 探询并标注中介性B（信念），识别其非理性特征
- 运用质疑、证据检验、替代视角等技术促发领悟
- 协同构建合理信念，填入A-B-C三栏表
- 确认至少3组条目达成双方共识并存档

## Constraints

- 须在建立基本咨访关系与安全氛围后启动
- 每次会谈聚焦1–2组清晰可溯的A-B-C链条
- B的识别需源自求助者原话，避免 therapist 预设归因

## Cautions

- 避免过早挑战信念引发阻抗；优先共情情绪，再探信念
- 不将‘合理’等同于‘积极’，而强调弹性、现实与自我关怀
- 若求助者难以命名B，可先用行为实验或情绪日记辅助觉察

## Output Contract

- 双方共同签署的A-B-C三栏表文档，含至少3组完整条目：每组明确列出A（具体可观察事件）、C（情绪+行为+生理反应）、原始B（原话引述）及修正后合理B（可验证、柔性、自我支持）

## Example Therapist Responses

### Example 1

- Client/Input: 求助者说：'这次小组汇报没被表扬，说明我彻底不行了，以后肯定没人愿意跟我合作。'
- Therapist/Output: A：小组汇报未获口头表扬；C：羞愧、手心出汗、回避后续讨论；B：'没被表扬=彻底不行'（过度概括）→ 合理B：'一次反馈不等于能力定论，我可以请同伴具体反馈改进点。'
- Notes: 示范如何从绝对化语言中提取B，并转向可操作、具身的合理信念

## 子技能目录
- [ABC理论引导与非理性信念识别](心理咨询/Family技能/认知行为疗法/二级技能/ABC理论引导与非理性信念识别/SKILL.md) ｜ 适用：在咨询中向求助者介绍合理情绪疗法的ABC理论，帮助其理解情绪和行为反应源于对事件的信念而非事件本身，并系统识别其绝对化、以偏概全等非理性信念； used early in anxiety treatment to externalize cognitive mediation and establish shared conceptual framework for belief identification.
- [ABC理论引导式叙事重构](心理咨询/Family技能/认知行为疗法/二级技能/ABC理论引导式叙事重构/SKILL.md) ｜ 适用：在咨询关系建立期，通过开放式提问唤起求助者关键生活事件记忆，辅以中性隐喻故事自然过渡至ABC理论框架，协助求助者自主识别诱发事件（A）、不合理信念（B）与情绪结果（C）之间的因果联结，不挑战、不修正B，仅完成命名与影响确认。
- [情绪ABC信念辨析与协同重构引导](心理咨询/Family技能/认知行为疗法/二级技能/情绪ABC信念辨析与协同重构引导/SKILL.md) ｜ 适用：运用情绪ABC理论引导来访者识别不合理信念，区分事件（A）、信念（B）与情绪反应（C）之间的逻辑关系，并通过苏格拉底式提问、证据搜寻与协同建构，发展更具现实性、灵活性与宽容度的替代性信念；强调以‘现实、理性、宽容’为标准检验信念功能，避免将理性等同于情绪压抑，同步接纳情绪体验的合理性；在巩固阶段进一步锚定求助者人际互动中的具体优势行为，将其作为反驳核心信念的具身化反例，促进认知弹性与功能迁移。本 skill now explicitly supports early-session automation-focused work: when core belief is not yet named but automatic thoughts are salient and event-linked, it includes structured event review, spontaneous thought capture, core belief linkage probing (‘如果这个想法是真的，那说明你是个怎样的人？’), multi-explanatory attribution generation, and micro-behavioral preheating (e.g., ‘团委会发言→我准备了，可以试一次’), all anchored in co-filled cognitive structure tables and grounded in real-life context — while preserving all original constraints, cautions, and output requirements.

## 选用规则（二级技能目录）
- 当目标、阶段或方法更接近 `ABC理论引导与非理性信念识别` 时，优先调用它。 线索：求助者已建立基本咨访关系, 存在明显情绪困扰但归因外化, 需提升对自动思维与信念关系的理解, client presents with emotion-behavior mismatch, client struggles to articulate belief-emotion links
- 当目标、阶段或方法更接近 `ABC理论引导式叙事重构` 时，优先调用它。 线索：求助者存在长期情绪困扰但归因模糊, 童年经历被确认为问题核心诱因, 需降低防御、提升理论接受度, ABC理论, 合理情绪疗法
- 当目标、阶段或方法更接近 `情绪ABC信念辨析与协同重构引导` 时，优先调用它。 线索：核心不合理信念已被清晰命名, 求助者具备基本抽象思维和反思意愿, 当次咨询时间充裕（≥45分钟）且无危机干扰, 来访者报告近期具体事件并伴随自我贬低想法, 已建立初步信任关系
