- quote: # 7.1第一阶段：建立咨询关系

第1次心理咨询，咨询师通过摄入性会谈给予来访者充分的尊重，建立了良性和稳定的咨访关系。首先，咨询师让其描述人际关系、自我的认知状态、适应能力、学习和娱乐等遇到的问题以及回顾她个人的成长史。其次，咨询师关注来访者的成长阶段是否发生过创伤性心理事件。第一次咨询结束之前，咨询师与来访者协商咨询目标和约定后续的咨询时间。

# 7.2第二阶段：实施个体心理咨询方案

在第 $2 \mathord { \sim } 6$ 次会谈中,咨询师与来访者商定和调整咨询方案并实施[。由于来访者多种记忆相互叠加，咨询师获得的信息都局限于她精神生活的表象，而处于表象之下的过去和当前经验一起构成了有机整体，但来访者只将其当前呈现的状态看成是心理问题的全部[4]。

根据《操作化心理动力学诊断和治疗手册》[]，咨询师将注意力放在来访者在参与的校内和校外活动的情绪变化上。通过识别来访者的社会支持来源(来访者的父母、男朋友、同学、舍友、老师和辅导员),咨询师针对其恐惧感、疏离感、厌恶孤独、感觉自己不讨人喜欢的部分，通过将无意识提升为意识化的表达，逐渐整合内在和外在力量，增进来访者的自我意识，改善其人际关系。
