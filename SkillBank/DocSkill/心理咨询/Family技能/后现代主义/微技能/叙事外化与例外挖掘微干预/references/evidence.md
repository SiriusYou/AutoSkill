1.2干预思路
以个案的咨询目标为导向，以叙说故事－问题解构-寻找例外－重构故事为思路，串联充满挫折、创伤和绝望的“问题故事”；再用外化技术动摇关于自杀的假设，重整问题经验，后寻找“独特事件”重叙“人生故事”以修正自我认识，实现自我认同。共进行7次叙事干预，每周1次，每次 $1 . 5 \sim 2 \mathrm { ~ h ~ }$ ，时间根据叙事进程和内容灵活调整。干预结束后1个月和2个月进行2次追踪。从自杀意念出现频率、心理状况、咨询进展及个案主观报告4个途径探究叙事干预的效果。为保证干预效果，叙事咨询师每2周接受1次专家督导。

1.3.1自杀意念出现频率评估
采用患者健康问卷抑郁症状群量表（PatientHealth Questionnaire 9-Itemdepression module,PHQ-9）@的条目9（以下简称为“PHQ-9")“在过去半个月里，有不如死掉或用某种方式伤害自己的念头”，评估自杀意念出现频率，采用4级评分（ $0 =$ 完全不会； $1 =$ 少数几天； $2 =$ 一半以上的天数； $3 =$ 几乎每天）。 $\mathrm { P H Q } \mathrm { - } \mathrm { ~ } 9$ 已被广泛应用于评估自杀意念的发生率，是自杀意念初始筛查的有效工具。本研究中，在每次叙事干预后，使用题干修改为“在过去的一周里”的PHQ-9测量自杀意念的出现频率，而1个月和2个月追踪则根据原题干的时间范围测量自杀意念的出现频率。
