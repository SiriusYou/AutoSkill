# EARS例外探索四步法

系统性引导来访者发现、扩展、强化并重复探索生活中的例外经验，通过Elicit-Amplify-Reinforce-Start again四步骤，催化其对自身资源与能力的觉察与调用。

## Prompt

当来访者报告任何好转迹象（无论多小）时，依次执行：① Elicit——用开放式问句邀请其具体描述‘发生了什么不同’；② Amplify——通过细节追问（如时间、人物、行为、感受）扩展该变化的情境与意义；③ Reinforce——以肯定性语言命名其自主性、能力或资源（如‘你主动选择了不还口，这需要克制和判断力’）；④ Start again——自然过渡回Elicit，询问‘还有别的不同吗？’或‘上次之后，有没有哪怕一小点新尝试？’。全程保持好奇、非评判语调，禁用解释、建议或归因。

## Objective

系统性放大来访者已发生的积极变化与自主行动
## Applicable Signals

- 来访者说‘好像好了一点’‘试了下没发脾气’‘同学今天没再推我’
- 来访者主动提及某次未重复旧模式的行为
- 会谈中出现微弱但可捕捉的积极动词（如‘我停了一下’‘我走开了’）

## Contraindications

- 来访者尚未识别任何例外或好转
- 当前会谈需处理突发危机或安全议题
- 咨询师尚未完成目标共构与奇迹问句澄清

## Intervention Moves

- Elicit: ‘能说说那次是怎么发生的吗？当时你在做什么？’
- Amplify: ‘谁在场？你说了什么？他们反应如何？你自己感觉怎么样？’
- Reinforce: ‘这说明你已经在调动自己的方法了’‘你选择暂停，是给自己留出了空间’
- Start again: ‘除了这个，还有没有其他时候，你感觉有点不一样？’

## Workflow Steps

- 确认并命名来访者报告的微小变化（Elicit）
- 通过细节追问深化其对自身行动的觉察与叙事厚度（Amplify）
- 将该行动与来访者内在资源、能力或价值观联结（Reinforce）
- 以开放姿态重启探索，延续正向线索（Start again）

## Constraints

- 不跳过Elicit直接评价
- 不将‘放大’变为过度追问施压
- 不将‘强化’替换为泛泛表扬（如‘真棒’）
- 不中断循环提前转入新话题

## Cautions

- 避免在来访者情绪仍高度脆弱时强行启动EARS
- 若来访者反复否认有任何好转，退回奇迹问句或评量问句，不强求EARS执行

## Output Contract

- 来访者能清晰复述至少1项自己已采取的具体行动，并表达对该行动效果的初步认可

## Example Therapist Responses

### Example 1

- Client/Input: 上次聊完，我有天没跟他们吵。
- Therapist/Output: Elicit: ‘那天发生了什么，让你没吵起来？’ → Amplify: ‘当时你在想什么？手有没有握紧？后来怎么让自己停下来？’ → Reinforce: ‘你调用了自己‘先停一下’的能力，这很关键。’ → Start again: ‘还有没有别的时间，你也这样‘停了一下’？’
- Notes: 聚焦身体信号与即时决策，避免抽象归因

### Example 2

- Client/Input: 我昨天放学绕路走了。
- Therapist/Output: Elicit: ‘绕路这个决定，是怎么做出来的？’ → Amplify: ‘你提前就计划好了？还是走到路口才改主意？路上遇到人了吗？’ → Reinforce: ‘你用行动保护了自己的安全距离，这是很实在的自我照顾。’ → Start again: ‘除了绕路，最近还有没有其他小调整？’
- Notes: 将回避行为重构为自主策略，锚定具体动作

## Objective

激活来访者内在资源与例外经验
## Applicable Signals

- 来访者使用‘有时候…’‘其实有次…’等隐含例外的语言
- 来访者自发提及未恶化时段或微小应对行为
- 情绪状态评量分数出现小幅波动（如从2分升至3分）
- 来访者说‘好像好了一点’‘试了下没发脾气’‘同学今天没再推我’
- 来访者主动提及某次未重复旧模式的行为
- 会谈中出现微弱但可捕捉的积极动词（如‘我停了一下’‘我走开了’）

## Contraindications

- 来访者尚无法区分问题与自我（需先完成外化对话）
- 例外叙述引发强烈羞耻或解离反应（需暂停并做稳定性处理）
- 来访者尚未识别任何例外或好转
- 当前会谈需处理突发危机或安全议题
- 咨询师尚未完成目标共构与奇迹问句澄清

## Intervention Moves

- Elicit: ‘能说说那次是怎么发生的吗？当时你在做什么？’
- Amplify: ‘谁在场？你说了什么？他们反应如何？你自己感觉怎么样？’
- Reinforce: ‘这说明你已经在调动自己的方法了’‘你选择暂停，是给自己留出了空间’
- Start again: ‘除了这个，还有没有其他时候，你感觉有点不一样？’

## Workflow Steps

- Elicit：用中性、好奇语气引出例外事件，避免预设或评判；确认并命名来访者报告的微小变化。
- Amplify：通过5W1H式细节追问，锚定例外的真实性与主体性；通过细节追问深化其对自身行动的觉察与叙事厚度。
- Reinforce：将例外行为与来访者身份特质（如责任感、共情力、毅力）明确联结，并给予具身化肯定（点头、前倾、停顿）；将该行动与来访者内在资源、能力或价值观联结。
- Start again：以轻柔过渡语开启新一轮例外探索，维持节奏与安全感；以开放姿态重启探索，延续正向线索。

## Constraints

- 不将例外解释为‘问题缓解’，而聚焦来访者主动参与和能动性
- 不跳过任一步骤，尤其不可省略Reinforce直接进入Start again
- 若来访者仅给出模糊回答（如‘好像好一点’），须退回Amplify层追问具体画面
- 不跳过Elicit直接评价
- 不将‘放大’变为过度追问施压
- 不将‘强化’替换为泛泛表扬（如‘真棒’）
- 不中断循环提前转入新话题

## Cautions

- 避免将‘例外’等同于‘应该多这样做’，防止隐性指责
- 当来访者将例外归因为外部因素（如‘那天没人烦我’），需温和探询其内在角色（‘那时你为自己做了什么来保护自己？’）
- 若连续两次Start again未引出新例外，暂停EARS，回归目标澄清或情绪稳定化
- 避免在来访者情绪仍高度脆弱时强行启动EARS
- 若来访者反复否认有任何好转，退回奇迹问句或评量问句，不强求EARS执行

## Output Contract

- 来访者能自主描述至少一个具体例外事件，并关联自身能力（如‘那次我没割手，是因为我想到小说还没写完’）
- 来访者能清晰复述至少1项自己已采取的具体行动，并表达对该行动效果的初步认可
