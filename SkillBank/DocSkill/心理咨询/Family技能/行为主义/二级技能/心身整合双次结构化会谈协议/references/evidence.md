# 3.3咨询次数与内容
一共两次，总共时长为3h。第一次会谈主要是与父母的会谈，时长为1h；第二次为对来访者单独进行咨询以及帮助父母来访者之间进行沟通，共2h。咨询完成后1个月内以线上咨询的方式针对来访者的实际情况对家长进行指导。

第一次咨询主要就来访者的问题与其父母进行会谈，并要求父母带上来访者的体检报告，主要了解来访者的生理状况、情绪状况、家庭环境、学习压力。后结合来访者体检报告向父母解释，因为来访者处于童年向成年过渡期，生理发育迅速，而心理发育相对缓慢，身心处于不平衡的状态。且青少年对自己生理和心理的快速变化、成长的烦恼、学校和家庭的期望、社会的巨大变革的适应很不容易，所以容易产生适应危机[5]。让父母了解到来访者是由于巨大的学业压力产生失眠，长时间失眠则导致白天精神不济、情绪波动大、记忆力减退，并不是来访者在装病。此外，告知来访者父母来访者在服用精神类药物的过程中产生的一些不良反应，如头晕、嗜睡、精力不济等，对家长进行药物服用以及缓解药物不良反应的指导。此次咨询不仅是为了了解来访者的信息，反过来也能让家长更好地明白来访者的处境和状况、增强其对解决问题的信心、方便后续的咨询和配合。

第二次咨询来访者由母亲和姨妈带来，告诉来访者咨询时可以选择家属在场与否，但事后会把来访者愿意的事情告知父母，来访者不愿意的事情无论当面或者私下都不会告诉其父母，希望来访者日后可以亲自告知，后来访者选择由姨妈陪同咨询。开始以介绍词根词缀法记忆单词为例，告诉来访者很多时候学习是有方法的，再引出原型物的单词，告知可以通过沙盘上原型物的摆放可以象征性的表达我们的内在世界。通过沙盘，发现来访者有喜欢对抗权威的倾向，以此为出发点询问其对父亲、学校老师的态度，得知来访者不喜欢被束缚、喜欢自由，在家被事无巨细管束太多觉得很难受、自己的想法得不到重视、常常因为这些事和家里人产生矛盾，后询问对家庭期望得到怎样的帮助以及希望家庭如何改变。然后询问来访者最为头疼的学习问题，帮助其根据目前自身情况和当地学校中考分数线确立清晰的中考目标。目标划分为3级：可接受目标、可达到目标和理想目标，自身稍加努力即可达到为可达到目标，可达到更进一步为理想目标，发生意外达到的为可接受目标。例如英语的词根词缀记忆法，好好总结各科学习规律，日常将成绩保持在可达到目标之上，努力去博最终理想目标，能达到最好，不能达到也在自己可接受范围内。用最适合自己的目标相比较来代替同学之间盲目的学习成绩攀比，帮助自己合理定位，减缓学习压力。最后和来访者交流他的身体健康问题，由于学业压力过大引起的失眠、生活方式不良以及服用精神类药物所带来的不良反应，致使他睡眠状况不佳、日常精力不足、情绪起伏较大、体力不支难以正常学习。为了动力不足的问题，调整其原有生活方式、饮食清淡、多吃蔬菜水果，坚持定期定量运动排汗对促进血液新陈代谢，增强体质。运动完之后进行泡澡，时长约为 $3 0 { \sim } 4 5 \ \mathrm { m i n }$ ，放松肌张力，将生理调整至放松状态以对抗焦虑。

待确定来访者暂无疑问后，邀请来访者母亲进入咨询室引导双方进行交流。先大概和来访者母亲交流来访者允许的内容，了解来访者的内心世界，再委婉地提出来访者对家庭的期待与要求，但无论如何，始终要坚持一条原则：与来访者身体健康、成长发展相关的事项，必须在父母的管理范围内，其余事情，可充分征求、尊重来访者的意愿。
